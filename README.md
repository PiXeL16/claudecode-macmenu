## Development

```bash
# Install dependencies
npm install

# Run in development mode
npm run dev

# Build TypeScript
npm run build

# Package for distribution
npm run package
```

## Requirements

- macOS 10.13 or later
- Node.js 18 or later

## Usage

The app runs in your menu bar and monitors Claude Code activity. Click the menu bar icon to:
- View session statistics
- Configure notification sounds
- Access preferences
- Quit the application

## Configuration

Settings are stored in: `~/Library/Application Support/claudecode-macmenu/`
